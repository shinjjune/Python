# module?

모듈이란 함수나 변수 또는 클래스들을 모아 놓은 파일이다.
모듈은 다름 파이썬 프로그램에서 불러와 사용할 수 있겠끔 만들어진 파이썬 파일

import 모듈이름

```
from 모듈이름 import 모듈함수
```
